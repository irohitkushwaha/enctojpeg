# EncToJPG - Secure .enc to JPG/PNG Converter

> **Live Tool:** [https://enctojpg.online](https://enctojpg.online)

**EncToJPG** is a free, secure, and privacy-focused tool designed to convert `.enc` files (commonly found as WhatsApp Web profile pictures) into viewable image formats like **JPG**, **JPEG**, and **PNG**.

## üöÄ Why EncToJPG?

When you save a profile picture from WhatsApp Web, it often downloads with an `.enc` extension, which is encrypted or encoded and cannot be opened by standard image viewers. **EncToJPG** solves this problem instantly.

### Key Features
- **Client-Side Conversion**: All processing happens entirely within your browser. Your files serve **NEVER** leave your device.
- **Privacy First**: No file uploads to any server. 100% secure.
- **Instant Result**: Drag and drop your file and get the converted image immediately.
- **Multiple Formats**: Support for converting `.enc` to both JPG and PNG.
- **Responsive Design**: precise and premium UI that works on desktop and mobile.

## üì¶ Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/irohitkushwaha/enctojpeg.git
   cd enctojpeg
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
